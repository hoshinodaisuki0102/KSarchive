# KSarchive 메인 페이지

밝고 쨍한 블루/하늘/화이트 톤의 KSarchive 메인 페이지입니다.  
기존 자바아카이브/한국사 사이트와 다른 방향으로, 더 신식이고 대시보드스러운 분위기로 설계했습니다.

## 실행 방법

```bash
npm install
npm run dev
```

## 포함 내용
- 메인 히어로 섹션
- 과목 6개 카드
- 시험 범위 요약 위젯
- 최근 업데이트 위젯
- 전체 AI 질문 허브 섹션
- 과목별 AI 진입 구조 반영
- 퀴즈 시스템 소개 섹션
- 플로팅 AI 버튼

## 다음 단계
다음에는 네가 과목 내용을 한 과목씩 보내주면:
- `/subjects/[subject]` 상세 페이지
- 시험 범위 데이터
- 실제 퀴즈 데이터 구조
- AI 채팅 연결용 구조

까지 이어서 맞출 수 있습니다.

## 국어 교과서 업데이트
- `/subjects/korean` : 국어 교과서 범위 페이지 추가
- 작품: `배를 매며`, `봄봄`
- 선생님 필기 / 아카이브 보충 / 모두 보기 토글 추가
- 지문 전체를 볼 수 있도록 교과서 페이지 이미지 전체 삽입
- 핵심 분석은 지문 사이에 공책 정리 스타일로 삽입


## 본문 정리 강화 업데이트
- 영어 본문을 문단 블록 + 문단별 필기 구조로 재정리
- 영어 기본 탭 진입 시 단어가 아니라 본문 탭이 먼저 열리도록 수정
- 영어 모의고사 18~45 주요 문항의 실제 지문 텍스트를 보강
- 국어 `배를 매며`, `봄봄`을 문단 단위 스크롤형 본문 + 중간 필기 구조로 재정리
- 국어 시는 연별 줄내림이 살아 있도록 수정
